# task-management-system
A full-stack task management web application built using Flask, JavaScript, RESTful APIs, and SQLite.
This project is a full-stack task management application that allows users to manage tasks efficiently.
It integrates backend APIs with a dynamic frontend interface.

Technologies used:
Python, Flask, JavaScript, SQLite, REST APIs

Features:
	•	Full CRUD operations
	•	RESTful API integration
	•	Dynamic frontend
	•	Task tracking system
